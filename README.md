# Risamaki-Chat

A simple chat application made using NodeJS and MongoDB

Functionality: 
* Make an account 
* Sign up with a previously made account
* Login / Sign up and go to chat window
* Logout


List of pages:
* Home Page
* Login
* Logout
* Sign up
* Chat Window 
* Reset password page

Deployed on Heroku: https://risamaki-chat.herokuapp.com
